1. システム概要

1.1 目的

本システムは、IT業務における作業実績を日々記録し、月次で集計・報告するためのツールです。タスク単位で作業時間を計測し、プロジェクトやカテゴリ別に集計することで、効率的な作業報告を実現します。

1.2 現状と課題

  • 現状：PostgreSQLに接続したシンプルなタスク管理ツール（登録・一覧・削除のみ）
  • 課題：時間計測機能、カテゴリ分類、集計機能、印刷機能が未実装

1.3 改修の方針

  • 既存のFlask + PostgreSQL構成を活用
  • タイマー機能によるタスク実行時間の自動計測
  • 月次・カテゴリ別の集計機能
  • A4印刷対応のレポート出力

3. データベース設計

3.1 テーブル定義

テーブル名 tasks（作業タスクテーブル）
カラム名 型 NULL 説明
id SERIAL NOT NULL 主キー（自動採番）
task_name VARCHAR(100) NOT NULL タスク名（プロジェクト名）
category VARCHAR(50) NOT NULL カテゴリ（開発、会議、メール、調査、その他）
memo TEXT NULL メモ（任意）
start_time TIMESTAMP NULL 作業開始時刻
end_time TIMESTAMP NULL 作業終了時刻
duration_seconds INTEGER NOT NULL 作業時間（秒単位）、DEFAULT 0
created_date DATE NOT NULL タスク登録日（年月日）
created_at TIMESTAMP NOT NULL レコード作成日時、DEFAULT NOW()

3.2 主な集計SQL例

-- プロジェクト別月次集計
SELECT
    task_name,
    SUM(duration_seconds) / 3600.0 AS total_hours,
    COUNT(DISTINCT created_date) AS work_days,
    ROUND(SUM(duration_seconds) * 100.0 /
        (SELECT SUM(duration_seconds) FROM tasks
         WHERE DATE_TRUNC('month', created_date) = '2020-01-01'), 1) AS percentage
FROM tasks
WHERE DATE_TRUNC('month', created_date) = '2020-01-01'
GROUP BY task_name
ORDER BY total_hours DESC;

-- カテゴリ別月次集計
SELECT
    category,
    SUM(duration_seconds) / 3600.0 AS total_hours,
    COUNT(DISTINCT created_date) AS work_days,
    ROUND(SUM(duration_seconds) * 100.0 /
        (SELECT SUM(duration_seconds) FROM tasks
         WHERE DATE_TRUNC('month', created_date) = '2020-01-01'), 1) AS percentage
FROM tasks
WHERE DATE_TRUNC('month', created_date) = '2020-01-01'
GROUP BY category
ORDER BY total_hours DESC;
